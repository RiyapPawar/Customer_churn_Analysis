# 📊 Customer Churn Analysis Dashboard | Power BI

## 📌 Project Overview

The **Customer Churn Analysis Dashboard** is an interactive Power BI project designed to analyze customer churn patterns and identify the key factors influencing customer retention. This dashboard transforms raw customer data into meaningful business insights through interactive visualizations and KPIs.

The project helps stakeholders understand customer behavior, monitor churn trends, and support data-driven business decisions.

## 🎯 Objectives

- Analyze customer churn trends.
- Identify customer segments with high churn.
- Compare churn across states, age groups, and tenure groups.
- Understand the impact of internet services, payment methods, and contracts on customer churn.
- Provide actionable insights for improving customer retention.

## 📈 Dashboard Features

### KPI Cards
- Total Customers
- New Joiners
- Total Churn
- Churn Rate

### Visualizations
- Customer Churn by Gender
- Churn Rate by Age Group
- Churn Rate by Tenure Group
- Churn Rate by State
- Churn Rate by Internet Type
- Churn Rate by Payment Method
- Churn Rate by Contract Type
- Total Churn by Category
- Service-wise Customer Analysis

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Cleaning
- Data Visualization

## 📊 Key Insights

- Identified the customer groups with the highest churn rate.
- Compared churn across different states.
- Analyzed churn based on internet service type.
- Evaluated the influence of payment methods and contract types.
- Identified major churn categories affecting customer retention.
- Created an interactive dashboard for business decision-making.
